# EPower Network Zealot Project Block

EPower Network Zealot 项目的小组件，非业务代码

## 安装

~~~
composer require epnz/block
~~~

## 使用

~~~ php
<?php

require __DIR__ . '/vendor/autoload.php'; 

// 请求端IP
block()->request->ip()；

~~~
